available panels


HelloPanel

Welcomes the user with the project name, the version, the URL as well as the authors.

This panel welcomes the user by displaying the project name, the version, the URL as well as the authors.

There are also an HTMLHelloPanel (rendering html instead of text) and a CheckedHelloPanel (checking the registry for a previous installation).


InfoPanel

This is a kind of 'README' panel.

This is a kind of 'README' panel. It presents text of any length.

There is also an HTMLInfoPanel rendering html instead of text.


LicencePanel

Prompt the user to acknowledge a license agreement.

These panels can prompt the user to acknowledge a license agreement. They block unless the user selects the 'agree' option.

There is also an HTMLLicencePanel rendering html instead of text.


TargetPanel

Allows the user to select the installation path.

This panel allows the user to select the installation path.

The installation path can be customized and the "target directory will be created" dialog can be suppressed.


PacksPanel

Allows the user to select the packs he wants to install.

Allows the user to select the packs he wants to install.

The packs can be required/optionnal and be grouped into installGroups. There are also an ImgPacksPanel (each pack has a different picture), a TreePacksPanel (uses a tree-like list instead of a simple list).


InstallPanel

Launches the installation process.

You should always have this one as it launches the installation process !


SimpleFinishPanel

An ending panel.

An ending panel.

There is also a FinishPanel able to write automated installer information.
